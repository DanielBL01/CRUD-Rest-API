# CRUD REST API  
This is a simple and basic REST API built using gorilla/mux and Go (Golang)

## Install gorilla/mux
go get -u github.com/gorilla/mux

## Endpoints 
### Get all schools
```bash
GET /schools
```

### Get single school
```bash
GET /schools/{id}
```
### Create new school
```bash
POST /schools
```

### Delete single school
```bash
DELETE /schools/{id}
```

### Update single school
```bash
PUT /schools/{id}
```
